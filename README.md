Morse Code Converter is a simple Python application that converts normal text into Morse code.
The program reads Morse symbols from a CSV file, stores them in a dictionary, and converts user input into its corresponding Morse code output.
It supports alphabets, numbers, symbols, and handles spaces between words using `/`. 
This project is designed for beginners to understand Python CSV file handling, dictionary mapping, and string processing through a practical mini application.

The application uses Python and CSV file handling and is suitable for learning and small real-world practice projects.
